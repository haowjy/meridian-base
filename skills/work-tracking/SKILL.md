# Work Tracking

Anchor work in a work item. Active artifacts live in the work directory;
spawns reference the work item for continuity across sessions and compaction.
If understanding is not on disk, it does not survive.

Write findings, decisions, and design artifacts to the work directory as you
produce them. Reference files accumulate; conversation context compacts. The
work directory is the shared state between you, the user, and other agents.

Load /meridian-work-coordination for work item mechanics, /dev-artifacts for
file placement conventions.
